# NewsIntel AI — Stock Sentiment Dashboard

> Real-time financial news analysis with AI-powered BUY/SELL/HOLD signals

![Python](https://img.shields.io/badge/Python-3.10+-blue) ![AI](https://img.shields.io/badge/AI-Claude%20API-orange) ![License](https://img.shields.io/badge/License-MIT-green)

## What it does

- Analyzes **2,800+ financial news articles** per day
- Scores each article with AI sentiment (-1.0 to +1.0)
- Generates **BUY / SELL / HOLD signals** for US stocks
- Visualizes sentiment trends hour by hour
- Tracks top market topics in real time

## Quick Start

```bash

# Run the analyzer

```

- Real-time sentiment scoring
- RSI + Moving Average analysis
- Volume confirmation signals
- Multi-stock parallel analysis
- Clean exportable reports

- Avg holding period: 3-5 days
